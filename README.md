# Scrapbot Le Bon Coin

## Dépendances et librairies

Ce scrapbot utilise les SeleniumBase, BeautifulSoup et Json

Afin de les utiliser, il faut les installer, pour cela executez ces commandes (json est normalement installé de base) :

    pip install seleniumbase
    pip install beautifulsoup4

SeleniumBase utilise Google Chrome, il sera donc nécessaire de l'installer pour utiliser ce script

## Lancement du projet

Une fois que toutes les librairies et dépendances sont installées, vous pourrez utiliser ce projet.
Pour cela, clonez ce repo avec la commande :

    git clone https://github.com/gameking773/scrapbot.git

Puis pour lancer le script, executez :

    # Sous Windows    
        python Bot.py

    # Sous Linux / macOS
        python3 Bot.py
